## Desarrolla una aplicación de gestión básica de una empresa utilizando TDD. 

El TDD (desarrollo dirigido por tests) es una práctica de programación que consiste en escribir primero las pruebas (generalmente unitarias), después escribir el código fuente que pase la prueba satisfactoriamente y, por último, refactorizar el código escrito. 
<br><br>
La aplicación servirá para crear empleados, directores, trabajadores e ingenieros.<br>
 - Cada **empleado** tiene un nombre y un departamento. Los **directores**, **trabajadores** e **ingenieros** son un tipo de empleado.<br>
 - Cada **director** tiene informes.<br>
 - Cada **trabajador** tiene proyectos.<br>
 - Cada **ingeniero** tiene un lenguaje de programación y está en el departamento de ingeniería.
